Fitur Wiki: Survival & Command Essentials untuk Member

Panduan bertahan hidup dan command dasar untuk member (Essentials):

  • /spawn – Teleport ke spawn server
  • /home – Teleport ke rumah yang telah di-set
  • /sethome – Menetapkan lokasi rumah
  • /tpa – Mengirim permintaan teleport ke pemain lain
  • /back – Kembali ke lokasi sebelumnya
  • /kit – Mengambil kit untuk member
  • /msg – Mengirim pesan ke pemain lain
  • /reply – Membalas pesan terakhir

Panduan RedProtect

Untuk melindungi wilayah menggunakan RedProtect, ikuti langkah berikut:

  • Buat kotak wilayah menggunakan pager (alat khusus untuk menentukan area).
  • Pasang sebuah sign di pager dan ketik [rp] pada sign tersebut.
  • Gunakan perintah tambahan RedProtect jika diperlukan untuk mengatur flag dan izin wilayah.

Cara Bermain Minecraft

Berikut beberapa tips dan panduan bermain Minecraft ala fandom:

  • Pelajari dasar-dasar bertani, bertahan hidup, dan membangun.
  • Jelajahi dunia untuk menemukan resource berharga.
  • Bergabunglah dengan komunitas untuk berbagi pengalaman dan tips.
  • Coba mode permainan seperti survival, creative, dan adventure.
  • Manfaatkan command dasar untuk mempermudah eksplorasi dan permainan.
